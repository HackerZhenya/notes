# Notes

Тестовое задание для школы мобильной разработки NapoleonIT

# Настройка окружения

Чтобы запустить проект, нужно:


1. Установить зависимости

```shell
composer update
```

2. Выполнить миграции 

```shell
php artisan migrate
```

3. Запусить сервер

```shell
php -S localhost:8000 -t public
```
